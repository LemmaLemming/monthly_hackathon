###Healthline Dispatcher Transcription Interface
Created Healthcare Hackathon March 2026. Node.js
<img width="2560" height="1492" alt="image" src="https://github.com/user-attachments/assets/9f6a0b06-81cc-40e4-8c0d-b7ceaa584aca" />
This app was designed to expand 811 healthline capacity by allowing one qualified persons (Nurse Practitioners, Physicians, Doctors)
to monitor many streams of 811 calls at once. This client decision support system exists outside 
of existing CAD apps such as telehealth. Operators who have questions during the call can use Healthline to summarize live transcripts and message qualified persons. 
qualified persons interface allows instant feedback. An end call feature sends sms to patient for follow up instructions. 

\webapp is client interface and \phycisians-portal is the physician interface. **This app requires transcription software api key and gemini api key**
